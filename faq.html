Frequently Asked Questions

Find answers to common questions about our 7-day web development course

This course is designed for absolute beginners with no prior coding experience who want to learn web development from scratch. It's also suitable for those with some basic knowledge who want a structured, project-based approach to solidify their skills.

You'll need:

  • A computer (Windows, Mac, or Linux)
  • Internet connection
  • A modern web browser (Chrome, Firefox, or Edge recommended)
  • A code editor (we recommend VS Code which is free)
  • No prior coding experience required!

By the end of the 7 days, you'll have built a complete, responsive personal portfolio website that includes:

  • Multiple sections (About, Projects, Contact)
  • Responsive design that works on all devices
  • Interactive elements with JavaScript
  • A contact form
  • Your own custom domain (we'll show you how to set this up)

We recommend setting aside 1-2 hours per day to complete each lesson. The course is designed with busy schedules in mind:

  • Day 1: 1 hour (setup and fundamentals)
  • Days 2-4: 1.5 hours each (HTML, CSS, responsive design)
  • Days 5-6: 2 hours each (JavaScript, building complete page)
  • Day 7: 1 hour (deployment and sharing)

You can go at your own pace - the 7-day timeline is just a suggested schedule.
