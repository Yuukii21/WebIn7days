Each day builds on the previous one, taking you from the basics to a complete website.

Introduction & Tools Setup

You'll Learn:

  • What exactly is a website?
  • Setting up VS Code editor
  • Understanding file structure
  • Essential web development tools

HTML Basics

Learn the foundations of HTML and structure your first webpage.

You'll Learn:

  • HTML document structure
  • Headings, paragraphs, and text formatting
  • Adding images and links
  • Creating lists and tables

CSS Basics

Add style and design to your webpage using CSS.

You'll Learn:

  • Colors, fonts, and text styling
  • Box model and layout basics

Responsive Design

Make your website look great on all devices.

You'll Learn:

  • Mobile-first design principles
  • Media queries for different screen sizes
  • Flexible layouts and units
  • Testing on multiple devices

Adding Interactivity

  • Handling click events

  • DOM manipulation basics
